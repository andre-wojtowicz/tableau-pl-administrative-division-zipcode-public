# Administrative division of Poland with zipcodes for Tableau

This project shows how Tableau geocoding can be extended for Poland by adding Poczta Polska zipcodes (kody pocztowe) along with voivodeships (wojew贸dztwa), counties (powiaty) and communes (gminy). A lookup table can also be made in order to transform Poczta Polska zipcodes to those recognized by Tableau.

* [CODGiK](http://www.codgik.gov.pl/index.php/darmowe-dane/prg.html) state registry of borders: 2017-01-27
* [Poczta Polska](http://kody.poczta-polska.pl/) zipcodes: 2017-05-01
* [Tableau](https://www.tableau.com/) zipcodes: 10.2
